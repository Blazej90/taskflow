# TaskFlow (Angular)

Mini task manager (To-do) built step-by-step to learn Angular.

## Goals

- Routing (`/tasks`, `/tasks/new`, `/tasks/:id/edit`)
- Components + templates
- Services + Dependency Injection
- HTTP + RxJS
- Reactive Forms
- Basic UI + filtering by status (todo/doing/done)

## Tech

- Angular
- TypeScript
- pnpm
